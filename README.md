# Fashion Store - Premium E-Commerce Website

## Overview
A complete frontend fashion e-commerce website with modern design, responsive layout, and full cart/wishlist functionality using localStorage.

## Features
- Responsive design (320px to 1440px+)
- Product filtering and search
- Shopping cart with localStorage
- Wishlist functionality
- Checkout with dummy payment
- Admin panel (password: admin123)
- 30+ dummy products
- Modern animations

## How to Run Locally
1. Clone or download the project folder
2. Open `index.html` in your browser
3. Or use Live Server extension in VS Code

## Admin Access
- Go to `/admin/login.html`
- Password: `admin123`

## Deploy on Netlify
1. Push to GitHub repository
2. Log into Netlify
3. Click "New site from Git"
4. Connect GitHub and select repo
5. Deploy (no build command needed)

## Future Backend Integration
Replace localStorage calls with API endpoints using Node.js/Express and MongoDB.
